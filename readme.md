## What I'm working on

A synthetic AI customer calls a Genesys-hosted voice AI, conducts multi-turn conversations, evaluates results, and logs everything. Built with Python, custom SIP server, Groq (LLM + STT), Piper TTS, raw UDP RTP with G.711 encoding, and Flask — fully containerized in Docker.

Maven multi-module monolith in Java with dedicated modules for auth, tickets, knowledge, AI, and analytics. Backed by Postgres/pgvector, Redis, Kafka (KRaft), and observed via Prometheus/Grafana.
